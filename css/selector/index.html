    CSS Selectors

    Normal selector

    <head>
    <link href="./style.css" rel="stylesheet">
</head>

<body>
    <h3>This is h3</h3>
    <p>Satheesh Pandian</p>
</body>


    h3 {
  color: blue;
}
p {
  color: red;
}
    In the above example, h3 and p are called as selectors as they are updated in CSS file and changing the property based on the value mentioned in CSS file.

    Class selector(.)

    You can select any HTML element as a normal selector

    <head>
    <link href="./style.css" rel="stylesheet">
</head>

<body>
    <h3>This is h3</h3>
    <p class="class-name">Satheesh Pandian</p>
</body>

    h3 {
  color: blue;
}
p {
  color: red;
}
.class-name{
    color: violet
}

    In the above example, though p element is defined in CSS file, class-name value has been applied due to high priority.

    class selector starts with . in the CSS file. You can apply class name to multiple HTML elements.

    <head>
    <link href="./style.css" rel="stylesheet">
</head>

<body>
    <p class="class-name">Satheesh Pandian</p>
    <h3>This is h3</h3>
    <h5 class="class-name">This is h5</h5>
</body>

    h3 {
  color: blue;
}
p {
  color: red;
}
.class-name{
    color: deeppink
}

    ID selector (#)

    <head>
    <link href="./style.css" rel="stylesheet">
</head>

<body>
    <h3>This is h3</h3>
    <p id="satheesh-id">Satheesh Pandian</p>
</body>

    h3 {
  color: blue;
}
p {
  color: red;
}
#satheesh-id{
    color: darkorange
}

    ID selector starts with # in the CSS file. This is unique, and you CANNOT apply id more than one HTML element.

    Attribute selector

    <head>
    <link href="./style.css" rel="stylesheet">
</head>

<body>
    <p class="class-name" draggable="true">Satheesh Pandian</p>
    <h3>This is h3</h3>
    <p draggable="true">This is draggable</p>
</body>

    p {
  color: red;
}
p[draggable] {
  color: darkgreen;
}
.class-name {
  color: deeppink;
}

    Here, draggable is an attribute. You can select any attribute

    All selector (*)

    <head>
    <link href="./style.css" rel="stylesheet">
</head>

<body>
    <p>Satheesh Pandian</p>
    <h3>This is h3</h3>
    <p draggable="true">This is draggable</p>
</body>

    * {
  color: darkmagenta;
}
p[draggable] {
  color: darkgreen;
}
h3 {
  color: deeppink;
}
